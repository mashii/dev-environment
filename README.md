# 初始化个人开发环境

# 1. 包含
1. bash、inputrc
2. screen
3. vim settings
4. 个人开发常用 `shell` 工具, 包含：`目录标签跳转`，`回收站`


## 2. 安装
git clone git clone https://github.com/mashii/dev-environment.git
cd dev-environment
./contrl install

> 注：vim 插件需要手动下载
> 1. 执行 `vim` 命令
> 2. 在 `vim` 内部执行 `:PlugInstall` 即可自动下载 `vim` 插件

## 3. 卸载
./control uninstall
